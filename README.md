# Smart Campus DX 클릭형 프로토타입

연수원 현장 운영업무의 디지털 전환 방향을 검증하기 위한 React 기반 클릭형 프로토타입입니다.

## 포함 기능

- 교육생: 객실, 일정, 식단, 버스 수요조사·예약, QR
- 운영자: 객실배정, 일정관리, 입퇴실, 시설관리, 버스관리
- 관리자: 운영현황, 통계, KPI, 자동 보고서

## 유의사항

- 화면과 데이터는 업무 검증을 위한 예시입니다.
- 실제 개인정보, 인증, 데이터 저장 및 내부 행정프로그램 연계는 포함하지 않습니다.
- 현업 인터뷰 결과에 따라 업무규칙과 화면을 보완합니다.
